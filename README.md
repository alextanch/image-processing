### Вопросы
1. **Цифровое изображение. Цветовые модели**
    - Цветовая модель XYZ
    - Хроматическая диаграмма
    - Цветовая модель RGB
    - Цветовая модель HSV
    - ColorJitter аугментация
2. **Арифметические и геометрические операции над изображениями**
    - Эквализация гистограммы изображения
    - Интегральное изображение
    - Преобразование координат при повороте и масштабировании изображений
    - Интерполяция изображений (методом ближайшего соседа и билинейная)
3. **Пространственная фильтрация изображений**
    - Двумерная корреляция и свертка
    - Сепарабельные свертки
    - Template matching
    - Box–фильтр
    - Гауссовский фильтр
    - Медианная фильтрация
    - Билатеральная фильтрация
    - Guided filtering
    - Повышение резкости с помощью фильтра Лапласа
4. **Частотные методы фильтрации изображений**
    - Прямое и обратное дискретное преобразования Фурье
    - Спектр Фурье при повороте и сдвиге изображения 
    - Теорема о свертке. Вычисление свертки с помощью преобразования Фурье
    - Теорема о свертке. Фильтрация изображения в частотной области
    - Фильтры низких частот (идеальный, гауссовский, Баттерворта)
    - Фильтры высоких частот (идеальный, гауссовский, Баттерворта)
5. **Пороговая сегментация и морфологическая обработка изображений**
    - Итеративный алгоритм подбора порога сегментации изображения
    - Сегментация изображения методом Оцу
    - Сегментация изображения с помощью  метода скользящего среднего
    - Морфологические операции эрозии и дилатации
    - Морфологические операции размыкания и замыкания
    - Морфологические операции утончения и утолщения
    - Построение остова (skeletonization) 
6. **Детекция границ и ключевых точек на изображениях**
    - Детектор границ Cобеля
    - Детектор границ Марра-Хилдрета
    - Детектор границ Кэнни
    - Детектор ключевых точек Харриса
    - Построение SIFT ключевых точек и их дескрипторов 
    - Сопоставление ключевых точек: ratio test, cross check test
